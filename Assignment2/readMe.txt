Assignment 2 - Running Instructions

To run the XML Parser:

1. Open terminal (or Command Prompt)

2. Navigate to the project folder

3. Use the following command to check a valid XML file:
   java -jar Parser.jar res/sample1.xml

4. Use the following command to check an invalid XML file:
   java -jar Parser.jar res/sample2.xml

The program will display whether the XML document is constructed correctly or show the errors found.
